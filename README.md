## Http Request Capture written in Python

### Usage
`sudo python3 http_request_capture.py --iface en0 --port 80`
#### arguments
- `--iface` - default `en0`
- `--port` - default `80`

Requirements:
* Python3
* [kamene](https://github.com/phaethon/kamene)
